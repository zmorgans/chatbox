# AI Debate Partner

Practice and sharpen your debate skills with an AI opponent that challenges your arguments across three rounds.
https://youtu.be/Ta_MwT1NR7U?si=dtyvDmuxSYXPZ9Cd
## Features

- **3-Round Debates**: Opening, Rebuttal, and Closing rounds
- **Multiple Stances**: Argue Pro, Con, or Neutral positions
- **6 AI Personas**:
  - Academic Scholar - Uses research and formal logic
  - Courtroom Lawyer - Cross-examines and finds inconsistencies
  - Philosopher - Questions assumptions and explores ethics
  - Investigative Journalist - Fact-checks with evidence
  - Devil's Advocate - Aggressively challenges every point
  - Socratic Mentor - Uses questions to expose weaknesses
- **NSDA-Style Feedback**: Get scored on Clarity, Logic, Evidence, and Structure
- **Debate History**: Save and review past debates
- **Export**: Download full transcripts as JSON

## Setup

1. Go to **Settings** (password: `debate123`)
2. Enter your Gemini API key (get one free at [Google AI Studio](https://aistudio.google.com/app/apikey))
3. Start debating!

## How to Use

1. Enter a debate topic on the Home page
2. Select your stance (Pro/Con/Neutral)
3. Choose an AI persona
4. Debate through 3 rounds
5. Get AI feedback and scores
6. Export or save your debate

## Tech Stack

- React + Tailwind CSS
- Base44 Platform
- Google Gemini AI (optional)
